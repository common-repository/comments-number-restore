Restore the comments number after the 2.5 upgrade.

== Description ==

After a 2.5 upgrade you can find the comments number of your post fixed to zero:
this plugin computes a comments count on each post and fix the issue.

The code has be taken from the wordpress forum (and modified).

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
